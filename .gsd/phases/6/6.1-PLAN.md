# Plan 6.1: Node.js Backend Dockerfile

## Objective
Containerize the Express backend and its SQLite database so it runs predictably on any host without requiring local Node.js installations.

## Context
- .gsd/ROADMAP.md
- backend/ (If exists)

## Tasks

<task type="auto">
  <name>Create Backend Dockerfile</name>
  <files>backend/Dockerfile</files>
  <action>
    - If `backend` does not exist, ignore (or create a placeholder).
    - If it does exist, create `Dockerfile` using `node:18-alpine` as the base image.

    - Set the start command to `node index.js` (or equivalent).

  <verify>cat backend/Dockerfile</verify>
