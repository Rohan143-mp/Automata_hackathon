# Plan 6.2: PC Hub Dockerfile

## Objective
Containerize the Python Hub (main_server.py) and its critical dependencies including Whisper and PyTTSx3.

## Context
- .gsd/ROADMAP.md
- hub/main_server.py

## Tasks

<task type="auto">
  <name>Create PC Hub Dockerfile</name>
  <files>hub/Dockerfile</files>
  <action>
    - Create `Dockerfile` in the `hub/` directory.
    - Use a stable Python base image (e.g., `python:3.10-slim`).
    - Note: PyTTSx3 often requires system-level packages like `espeak-ng` or `alsa-utils` to function on Linux. Include these in the `apt-get install` commands.
    - Copy the `requirements.txt` from Phase 1 / Phase 3, run `pip install`, and copy the source code in.
    - Set the entrypoint to `python main_server.py`.
    - Expose ports `81` (Glove WS), `82` (App Sync WS), and `8765` (Whisper WS).
  </action>
  <verify>cat hub/Dockerfile</verify>
  <done>Dockerfile is ready to build the PC Hub image with all necessary system libraries.</done>
</task>

## Success Criteria
- [ ] PC Hub can be run without installing python/pip locally.
